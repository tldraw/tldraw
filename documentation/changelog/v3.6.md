## v3.6.0

This release focuses on developer experience improvements with expanded helper functions and various bug fixes.

### Breaking changes

- `setEmbedDefinitions` transitioned from instance method to static method on `EmbedShapeUtil` class
- Removed `ExtractOptionalKeys` and `ExtractRequiredKeys` types

### Improvements

- Expand helpers available in actions and toasts overrides, including clipboard interactions, toasts, exports, printing utilities, and additional helper functions ([#5050](https://github.com/tldraw/tldraw/pull/5050))
- Add utility type creation for optional properties
- Refine grid snapping for notes after position updates

### Bug fixes

- Resolve copy/paste issues with multiple items and video handling
- Fix text clipping in scaled frames during export
- Correct concurrent export font interference
- Address long-press coordinate accuracy
- Fix unnamed frame export naming
- Improve custom embed initial rendering

[View release on GitHub](https://github.com/tldraw/tldraw/releases/tag/v3.6.0)
