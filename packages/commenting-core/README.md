# @tldraw/commenting-core

Server-side commenting logic for the tldraw SDK: write authorization for comment and comment-thread records, for use with a sync server's `authorizeRecord` option (see `TLSocketRoom` in `@tldraw/sync-core`). This package has no react or client-editor dependencies, so it is safe to import from workers and node servers.

See [tldraw.dev](https://tldraw.dev) for docs. License: see `LICENSE.md`.
