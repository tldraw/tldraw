The tldraw SDK has evolved across three major versions. This page summarises what each one can do, so you can pick the right one for a new project — or plan an upgrade for an existing one. Most users should start with **5.0**, which has the broadest extensibility surface and is where new features land. **3.x is no longer supported**: we recommend that anyone still on 3.x upgrade to 5.0. **4.x** continues to receive critical fixes only.

For full details on what shipped in each release, see the [release notes index](/getting-started/releases).

## Support status

| Version | Status           | What it gets                                        |
| ------- | ---------------- | --------------------------------------------------- |
| **3.x** | End of life      | No further updates. Upgrade recommended.            |
| **4.x** | Maintenance only | Critical fixes. No new features.                    |
| **5.0** | Active           | New features, fixes, and improvements. Recommended. |
