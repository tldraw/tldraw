Add a tldraw canvas in your React app in just 5 minutes.

You will be making this:  

embed-of-the-repo

You can follow the steps below, or clone the repo link-to-the-repo. 

## Installation

- Set up a React project however you normally do
- Install the tldraw library using this command:
```bash
npm install @tldraw/tldraw@beta
```

## Import Styles
To import fonts and CSS for tldraw:

- Create or edit a css file called index.css 
- Copy and paste this into the file:
```CSS
@import url("https://fonts.googleapis.com/css2?family=Inter:wght@400;500;700&family=Roboto+Mono:wght@400;700&display=swap");
@import url("@tldraw/tldraw/tldraw.css");

body {
  font-family: "Inter";
}
```

## Render Tldraw Component

To render the Tldraw component

- Import the ```<Tldraw />``` component from “@tldraw/tldraw”
- Import index.css
- To render a full screen canvas, wrap the Tldraw component in a div with the style attribute set to: ```{position: 'fixed', inset: 0}```
```javascript
import { Tldraw } from "@tldraw/tldraw";
import "./index.css";

export default function App() {
  return (
    <div style={{ position: 'fixed', inset: 0 }}>
      <Tldraw />
    </div>
  );
}
```
