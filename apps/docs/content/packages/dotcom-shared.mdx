`@tldraw/dotcom-shared` contains shared types and utilities for tldraw.com and its backend services. It includes the database schema, mutators, permissions, and common API types.

## Key components

- Zero schema definitions for users, files, and file state
- Mutators for optimistic updates
- Shared route and API types

## Usage

```typescript
import { tlaSchema } from '@tldraw/dotcom-shared'
```

## Key files

- packages/dotcom-shared/src/tlaSchema.ts - Zero schema
- packages/dotcom-shared/src/mutators.ts - Shared mutators
- packages/dotcom-shared/src/routes.ts - Route and API types
