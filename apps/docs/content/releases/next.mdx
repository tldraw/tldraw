## Bug fixes

- Fix a rendering artifact where nested clipped shapes (for example an image under multiple `getClipPath` ancestors) could show a stray triangular wedge. `intersectPolygonPolygon` now uses Sutherland–Hodgman clipping instead of sorting intersection vertices by angle around their centroid. ([#9001](https://github.com/tldraw/tldraw/pull/9001))
