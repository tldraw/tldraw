tldraw organizes your work into **files** (sometimes called boards or projects in other tools). Your files are listed in the sidebar. Every file has an owner — if you created it, you're the owner.

![The tldraw sidebar showing a list of files](/images/user-guides/files.gif)

## Creating a file

Click **New file** at the top of the sidebar. Give your file a name and press **Enter**. The file opens immediately and is ready to use.

## Opening a file

Click any file in the sidebar to open it.

## Renaming a file

Double-click the file's name in the sidebar. Type the new name and press **Enter** to save, or **Escape** to discard.

You can also open the file menu (hover the file and click the **⋯** icon) and choose **Rename**.

## Duplicating a file

Open the file menu and click **Duplicate**. A copy of the file appears in your sidebar.

## Deleting a file

Open the file menu and click **Delete**. Deleted files cannot be recovered, so be sure before you delete.

## Pinning a file

Open the file menu and click **Pin** to keep the file at the top of your sidebar. To unpin it, open the file menu again and click **Unpin**.
