tldraw files are shared by default. Anyone with the link can access your file — you control whether they can edit it or only view it.

## Getting a share link

Click the **Share** button, open the **Invite** tab, and click **Copy link**. Send that link to anyone you want to collaborate with. They don't need a tldraw account to access the file.

![Copying a share link from the tldraw Share menu](/images/user-guides/sharing.gif)

## Setting visitor permissions

In the **Invite** tab, use the **Anyone with the link** dropdown to choose:

- **Editor** — visitors can see and edit the file in real time. They can't rename or delete the file, but can freely modify its contents.
- **Viewer** — visitors can see the file in real time but cannot make any changes.

You can change this setting at any time. Connected visitors update immediately when you do.

## Making a file private

To stop sharing, open the **Invite** tab and toggle off **Share this file**. The file becomes private — only you can access it. Any visitors currently in the file are disconnected immediately.

To share again, toggle **Share this file** back on. The original link still works.

## Real-time presence

When collaborators are in the file, you'll see their cursors and names. Each person gets a unique color. Hover a cursor to see who it belongs to.

## Cursor chat

Press **/** to open a chat bubble on your cursor. Type a message and press **Enter** — it floats next to your cursor for everyone to see. Press **Escape** to dismiss it.
