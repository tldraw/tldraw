Draw nested closed shapes (contour lines) and sculpt them into a 3D terrain.

---

This example detects nested closed shapes, assigns heights based on nesting depth, and projects the flat drawing into a 3D terrain visualization using perspective projection. Draw concentric ellipses or freehand closed shapes, then click "Sculpt 3D" to see the result. Adjust elevation and tilt angle with the sliders.
