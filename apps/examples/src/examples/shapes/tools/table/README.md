The built-in table shape — a grid of editable rich-text cells.

---

Tables are a default shape. This example adds the table tool to the toolbar and
seeds a small table with a header row. Click a table to select it, click again to
drill into a cell, and double-click to edit. Each cell is its own record, so it is
individually styleable (select a cell and use the style panel), bindable, and
multiplayer-safe. Drag the interior edges to resize a column or row; drag the
selection corners to scale the whole table.
