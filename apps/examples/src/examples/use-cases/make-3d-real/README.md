Turn canvas notes and images into a live 3D asset generated by an AI model.

---

This is a "make real" for 3D. Drop notes, stickies, and images on the canvas describing an asset you want, select them, and click **Make 3D real**. The example serializes the selection into a prompt, sends it to Claude, and renders the returned [React Three Fiber](https://r3f.docs.pmnd.rs/) scene back onto the canvas inside a custom `model-3d` shape.

To iterate, select the generated asset together with a new note (for example "make it bigger and blue") and click again — the shape is regenerated in place with the previous code as context.

How it works:

- **Model3dShapeUtil** is a custom shape whose `component()` mounts an R3F `<Canvas>`. The generated code (a string of JSX defining a `Scene` component) is transpiled with Babel standalone and evaluated into a live component. A curated set of three.js and drei globals is injected as the sandbox.
- **makeReal3d** collects text (via `renderPlaintextFromRichText`) and images from the selection, packages them into a prompt, streams a completion from the Anthropic API, and creates or updates the `model-3d` shape.

You'll need an Anthropic API key — the example prompts for one and stores it in `localStorage`. This calls the API directly from the browser, which is fine for local experimentation but not for production: proxy the request through a server so your key isn't exposed, and don't `eval` model output you don't trust.
