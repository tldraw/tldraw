Demonstrate different arrow styling options.

---

This example shows all the styling properties available for arrows in tldraw:

- **Dash styles**: `draw` (hand-drawn), `solid`, `dashed`, `dotted`
- **Colors**: Various default tldraw colors
- **Sizes**: `s`, `m`, `l`, `xl` affecting stroke width and arrowhead dimensions
- **Fill styles**: `none`, `semi`, `solid`, `pattern` for arrowhead interiors
- **Scale**: Multiplier that affects overall arrow dimensions

The `draw` dash style is particularly distinctive with its hand-drawn, sketchy appearance. Each row in the grid demonstrates a different style property with multiple variations.
