Extend the default color palette with extra slots.

---

Add 12 extra colors to the default theme by passing a `themes` prop with the augmented palette. Once the color grid contains more than 12 colors, the style panel's color section becomes scrollable and a resize handle appears under the opacity slider — drag it to grow the grid, double-click to reset.
