Register a custom named theme and switch between themes.

---

You can register themes beyond `light` and `dark` by passing a `themes` prop with additional entries. Use the `theme` prop (or `editor.setTheme()`) to switch between them. This example adds a "my-brand" theme with custom colors and provides buttons to switch between all three themes.
