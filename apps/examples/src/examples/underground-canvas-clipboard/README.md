Copy and paste reimagined as a visible, spatial process with underground carrier agents.

---

Select some shapes and press Cmd/Ctrl+C to copy. Instead of an instant teleport, the selection folds into a tiny package that sinks into the canvas. Autonomous carrier agents patrol faint underground pathways and pick up the package. Press Cmd/Ctrl+V somewhere else and a carrier delivers the package to your cursor, unfolding it back into the original shapes.
