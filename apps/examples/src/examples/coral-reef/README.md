Draw a flat shape and pull it into a 3D coral reef structure

---

This example demonstrates a custom shape with an interactive pull handle. Draw a closed loop,
then drag the handle to extrude it into a recursively branching coral reef rendered on a
Canvas 2D context for smooth performance.
