Stack and pack shapes for automated layout using the editor API.

---

This example demonstrates how to use `stackShapes` and `packShapes` to programmatically arrange shapes on the canvas. Stacking arranges shapes in a line — either horizontal or vertical — with a consistent gap between them. Packing uses a bin-packing algorithm to arrange shapes into a compact grid layout. Select shapes and use the buttons to try each operation. You can specify a custom gap (in pixels) or omit it to let the editor detect the gap automatically from the current spacing.
